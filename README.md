# Python Labs

Python Labs is a repository dedicated to documenting my journey in learning Python from scratch to advanced mastery. 

The ultimate goal of this repository is to build and develop Machine Learning models through a deep understanding of the underlying logic and calculations. Instead of just running high-level frameworks without knowing what happens under the hood, this project focuses on writing the workflow manually using supporting libraries to fully master the model-building process.

## Repository Purpose
* **Core Fundamentals:** Practicing clean code, efficient syntax, and core Python programming paradigms.
* **Logic & Calculations:** Understanding and implementing the actual logic and data processing steps behind Machine Learning.
* **Model Development:** Building, training, and optimizing models in a Linux Ubuntu environment using supporting libraries.

## Tech Environment
* **Operating System:** Ubuntu Linux
* **Language:** Python 3.x
